## 🛠️ 项目矩阵 (Project Matrix)

### 1. 🎓 AI 生涯规划全景推演 (Expert Edition)
基于 **Kelly Mahler 内感受觉分离理论** 与 **ASI 感统临床模型**，构建的多维升学算法。
- **动态规划**：根据“逻辑-空间-语言”三大智能权重，自动生成 K-12 逆向推演时间轴。
- **择校矩阵**：覆盖美/英/中三轨，根据录取概率提供“冲/配/保”三级分层策略。

### 2. 🤖 具身智能 (Embodied AI) 可视化实验室
将复杂的机器人运动学与人体解剖学对齐。
- **人机对齐**：通过 IMU 传感器对齐前庭觉、编码器对齐本体感受。
- **强化学习可视化**：直观展示机器人如何在仿真环境（RL）中通过试错学习人类姿态。

### 3. 👩‍🔬 个人数字简历 (Interactive Resume)
集成在 `index.py` (原 `app.py`) 的全栈专家看板，展示硬核技术栈与教育哲学。
